SoundCloud Scraper API

Free & Powerful SoundCloud API - Search tracks, get playlists, streaming URLs, and more

Fast Response Anti-DDoS Protected No Daily Limit 99.9% Uptime

Overview

SoundCloud Scraper API adalah layanan gratis yang memungkinkan Anda mengakses data dari SoundCloud tanpa perlu official API key. Mendukung pencarian track, detail playlist, rekomendasi terkait, komentar, dan streaming URL.


Base URL: https://api.fazzcode.qzz.io
Version: v1.0.0
Methods: GET, POST

Authentication

API Key Required

Semua request wajib menyertakan API key. Anda bisa mendapatkan API key dari:

curl -X GET "https://raw.githubusercontent.com/fazzcode/api-keys/main/soundcloud-keys.json"

Cara Kirim API Key

MethodHeader/ParameterExample
HeaderX-API-KeyX-API-Key: your_api_key_here
Bearer TokenAuthorizationAuthorization: Bearer your_api_key_here
Query Parameterapi_key?api_key=your_api_key_here

Code Examples

JavaScript (Node.js)

const axios = require('axios')

const API_KEY = 'your_api_key_here'
const BASE_URL = 'https://api.fazzcode.qzz.io'

// Search tracks
async function searchTracks(query) {
  const response = await axios.get(`${BASE_URL}/api/soundcloud`, {
    params: {
      action: 'search',
      query: query,
      limit: 10,
      api_key: API_KEY
    }
  })
  return response.data
}

// Get track detail
async function getTrackDetail(trackId) {
  const response = await axios.get(`${BASE_URL}/api/soundcloud`, {
    params: {
      action: 'track',
      track_id: trackId,
      api_key: API_KEY
    }
  })
  return response.data
}

// Usage
searchTracks('iqro').then(console.log)
getTrackDetail(2258880134).then(console.log)

Python

import requests

API_KEY = 'your_api_key_here'
BASE_URL = 'https://api.fazzcode.qzz.io'

# Search tracks
def search_tracks(query):
    params = {
        'action': 'search',
        'query': query,
        'limit': 10,
        'api_key': API_KEY
    }
    response = requests.get(f'{BASE_URL}/api/soundcloud', params=params)
    return response.json()

# Get track detail
def get_track_detail(track_id):
    params = {
        'action': 'track',
        'track_id': track_id,
        'api_key': API_KEY
    }
    response = requests.get(f'{BASE_URL}/api/soundcloud', params=params)
    return response.json()

# Usage
print(search_tracks('iqro'))
print(get_track_detail(2258880134))
