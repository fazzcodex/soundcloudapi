📚 Contoh Request

GET /api/soundcloud?action=homepage&limit=10&api_key=YOUR_KEY

GET /api/soundcloud?action=search&query=iqro&limit=10&api_key=YOUR_KEY

GET /api/soundcloud?action=track&track_id=2258880134&api_key=YOUR_KEY
Detail track Iqro' - Raim Laode
GET /api/soundcloud?action=playlist&playlist_id=1734880683&api_key=YOUR_KEY
Detail playlist Hip Hop & Rap

📖 Cara Penggunaan

1. Dapatkan API Key

https://raw.githubusercontent.com/fazzcode/api-keys/main/soundcloud-keys.json

2. Kirim Request dengan Header:

X-API-Key: your_api_key_here

Atau dengan Query Parameter:

?api_key=your_api_key_here

Atau dengan Bearer Token:

Authorization: Bearer your_api_key_here
